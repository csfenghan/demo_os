# demo os
一个学习性质的宏内核OS

## feature
1. 支持多核
2. 支持多线程
3. 支持虚拟内存
4. 支持X86、ARM架构
5. 支持网咯设备
6. C++实现

## 详细安排
### 一、std基础库开发
本任务目标是开发一款不依赖系统调用的c++基础库，支持一些简单的数据结构，后续在此基础库基础上开发内核

#### 目标
1. 支持打印输出功能(只需给出一个注册接口即可，该接口的具体实现由具体的硬件驱动程序给出)
2. 支持vector、array、queue、unordered_map等常用数据结构。
3. 给出测试demo并通过测试
